Faronics ANTI-EXECUTABLE STANDAR README

Dokumen ini bukan pengganti Faronics Anti-Executable Standar buku petunjuk.

1. Klik dua kali file MSI yang tepat untuk memulai proses instalasi:
   AESTD_32-bit_en.msi dapat diinstal pada edisi 32-bit Windows XP SP2 
   (atau yang lebih baru) atau Windows Vista.

   AESTD_64-bit_en.msi dapat diinstal pada edisi 64-bit Windows XP SP2 
   (atau yang lebih baru) atau Windows Vista.


   Edisi untuk Windows Server 2003/2008 (dan juga Enterprise editions) dapat 
   di-download secara terpisah dari http://www.faronics.com.

2. Bila diminta untuk License Key, masukkan satu untuk menginstal sebuah edisi 
   lisensi penuh Anti-Executable Standar. Jika License Key tidak dimasukkan, 
   Anti-executable akan berjalan sebagai versi evaluasi dan akan mulai beroperasi 
   selama 30 hari setelah instalasi.

   Sebuah versi evaluasi kadaluarsa tidak akan melindungi sebuah mesin. Versi evaluasi 
   dapat dikonversi ke versi lisensi penuh dengan memasukkan License Key yang valid. 
   Lisensi Tombol dapat dibeli dari Faronics Faronics atau mitra. Lisensi khusus untuk 
   Standar (XP / Vista) atau Standar Server (Server 2003/2008) sistem operasi. Untuk 
   informasi lebih lanjut, silakan kunjungi http://www.faronics.com.

3. Restart komputer setelah instalasi selesai.

4. Setelah restart, akses Anti-executable pengaturan dengan menekan 
   tombol Shift dan mengklik dua kali ikon Anti-executable di Windows System Tray. 
   Atau, CTRL + ALT + SHIFT + F10 hotkey urutan dapat digunakan.

5. Konfigurasi Anti-executable seperti yang diinginkan. Sebuah Daftar Putih perlu dibuat 
   pada tab Daftar Putih dan Anti-executable perlu diaktifkan pada tab Status. Lihat 
   Anti-Executable Standar buku petunjuk atau informasi mengenai semua fitur.
